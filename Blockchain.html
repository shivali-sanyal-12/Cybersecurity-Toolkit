What is Blockchain?

Blockchain is a decentralized, distributed ledger that records transactions across many computers. It's immutable, transparent, and secure due to cryptographic techniques and consensus algorithms.

  • Each block contains data, a hash, and the hash of the previous block.
  • If the data changes, the hash changes – breaking the chain.
  • This structure makes blockchain tamper-resistant.
  • Used in cryptocurrencies, supply chain, voting systems, etc.
