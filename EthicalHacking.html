Ethical Hacking

Authorized hacking to find security flaws and fix them

Ethical hacking means authorized attempts to access systems to find vulnerabilities and fix them before malicious hackers exploit them.

Common Phases ‚ùì Click each phase to see description

  • Reconnaissance
    • Gathering information about the target to understand weaknesses.
  • Scanning
    • Using tools to identify vulnerabilities and open ports.
  • Exploitation
    • Attempting to exploit weaknesses to gain access.
  • Reporting
    • Documenting the findings and recommending fixes.

Example: Simple Port Scan Demo

Enter IP address and see if common ports 21, 22, 80, 443 are "open" or "closed" (simulated):

Sample Code Snippet


// Simple port scan simulation
function simulatePortScan(ip) {
  const ports = [21, 22, 80, 443];
  return ports.map(port => {
    // Random open/closed
    const open = Math.random() > 0.5;
    return { port, status: open ? 'open' : 'closed' };
  });
}
